====== ONLINE AUDIO FILE VALIDATOR  ======

Online audio file validator is an application which checks if audio file is broken - if contains sudden silence gap in signal.
Input
Application takes as input digital audio signal (.wav).
OUTPUT:
Table with information: file name, valid/invalid and number of first sample of silence gap if file is invalid

=== Installing ===

To install required packages run: pip3 install -r requirements.txt

=== Run ===

Run program by typing in terminal: python3 -m main [file_name.wav] where 'file_name' is name of recording

=== Steps ===

1. Upload audio file
2. Click Submit
3. Look at output table
